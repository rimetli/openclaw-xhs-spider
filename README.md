# 🍉 小红书爬虫可视化控制台 (XHS-Spider-UI)

本项目基于开源爬虫底层接口二次开发，提供了一个现代化的 **Web 可视化控制台**，无需修改代码，小白也能一键下载小红书无水印作品及评论数据。

## ✨ 核心亮点
* **开箱即用的 Web UI**：基于 Streamlit 构建，告别黑框框终端。
* **博主主页批量下载**：一键抓取指定博主所有图文/视频。
* **评论数据可视化导出**：一键解析笔记下的所有评论、二级回复、IP属地，并导出为 Excel。

## 🚀 快速启动
1. 安装 Python 和 Node.js。
2. 克隆本项目并安装依赖：`pip install -r requirements.txt` 和 `npm install`。
3. 在 `.env` 文件中配置你的小红书 Cookie。
4. 运行可视化界面：`streamlit run ui.py`。
